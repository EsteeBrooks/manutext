גיבורי חילים שבעה עשר אלף ומאתים יוצאי צבא למלחמה :
ושפם וחפם בני עיר חשם בני אחר : בני נפתלי
יחציאל וגוני ויצר ושלום בני בלהה : בני מנשה אשריאל
אשר ילדה פילגשו הארמיה ילדה את מכיר אבי גלעד
ומכיר לקח אשה לחפים ולשפים ושם אחתו מעכה ושם
השני צלפחד ותהיינה לצלפחד בנות : ותלד מעכה אשת
מכיר בן ותקרא שמו פרש ושם אחיו שרש ובניו אלם ורקם :
ובני אלם בדן אלה בני גלעד בן מכיר בן מנשה : ואחתו [//]
המלכת ילדה את איש הוד ואת אביעזר ואת מחלה : ויהיו
בני שמידע אחין ושכם ולקחי ואניעם :
ובני אפרים שותלח וברד בנו ותחת בנו ואלדעה בנו ותחת
בנו : וזבד בנו ושותלח בנו ועזר ואלעד והרגום אנשי גת
הנולדים בארץ כי ירדו לקחת את מקניהם : ויתאבל אפרים
אביהם ימים רבים ויבאו אחיו לנחמו : ויבא אל אשתו
ותהר ותלד בן ויקרא את שמו בריעה כי ברעה הית(ה) בביתו :
ובתו שארה ותבן את בית חורון התחתון ואת העליון ואת
אזן שארה : ורפח בנו ורשף ותלח בנו ותחן בנו : לעדן
בנו עמיהוד בנו אלישמע בנו : נון בנו יהושע בנו : ואחזתם
ומושבותם בית אל ובונתיה ולמזרח נערן ולמערב גזר
ובנתיה ושכם ובנתיה : ועל ידי בני מנשה בית שאן ובנתיה
תענך ובנתיה מגדו ובנותיה דור ובנתיה באלה ישבו בני
יוסף בן ישראל :
בני אשר ימנה
וישוה וישוי ובריעה ושרח אחתם : ובני בריעה חבר [ומלכ']
ומלכיאל הוא אבי ברזות : וחבר הוליד את יפלט ואת שומר
ואת חותם ואת שועא אחתם : ובני יפלט פסך ובמהל
ועשות אלה בני יפלט : ובני שמר אחי ודוהגה וחבה וארם :
ובן הלום אחיו צופח וימנע ושלש ועמל : בני צופח שוח
וחרנפר ושועל וברי וימרה : בצר והוד ושמה ושלשה ויתרן
ובארא : ובני יתר יפונה ופספה וארא : ובני עלה ארח וחניאל
ורציא : כל אלה בני אשר בית האבות ברורים גיבורי חילים ראשי
והתיחשם בצבא במלחמה מספרם עשרים וששה אלף :
ובנימן הוליד את בלע בכרו
אשבל השני ואחרח השלישי : נוחה הרביעי ורפא החמישי :
ויהיו בנים לבלע אדר וגרא ואביהוד : ואבישוע ונעמן
ואחוח : וגרא ושפופן וחורם : ואלה בני אחוד אלה הם
ראשי אבות ליושבי גבע ויגלום אל מנחת : ונעמן ואחיה
וגרא הוא הגלם והוליד את עזא ואת אחיהוד : ושחרים
הוליד בשדה מואב מן שלחו אותם [/] חושים ואת בערא
נשיו : ויולד מן חדש אשתו את יובב ואת צבייא ואת מישא
ואת מלכס : ואכי יעוץ ואת שביה ואת מרעא אלה ראשי
אבות : ומחשים הוליד את אביטוב ואת אלפעל : ובני
אלפעל עבר ומשעם ושמד הוא בונה את אונו ואת
לד ובנתיה : ובריעה ושמע המה ראשי האבות ליושבי
אילון המה הבריתו את יושבי גת : ואחיו ששק וירימות
